## Installation

To start using our project, follow these simple steps:

1. Install the CLI:
   ```bash
   npm install -g @mintlify/cli
   ```

2. Initialize your project:
   ```bash
   mintlify init
   ```

3. Start the dev server:
   ```bash
   mintlify dev
   ```

## Next Steps

Once you have your project running, you can start adding content to your MDX files.
